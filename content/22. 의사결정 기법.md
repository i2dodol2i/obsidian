#마음의기술

### 먼저 해결해야 할 결정 방해 요인

- 재정적 상황
- 감정적 상황([스트레스](20.%20마음챙김), 두려움, [충동성](18.%20시차%20기법), 자기 효능감, 너무 다양한 관심, 탐탁지 않은 선택지, 대동소이[^1])
- 타인의 영향을 받는다
- 심각하고, 되돌리기 어렵다
- 처음 겪는다
- 주변의 지지 없이 혼자 결정한다
- 결정할 일이 너무 많다
- 선택할 수 없거나 싫다[^2]

---

### 의사결정 기법

1. 문제에 대해 명확한 견해를 지니고 있는가? 근본 원인은 무엇인가?
   주변이나 전문가의 도움을 얻어야 한다, 단 의지해서는 안 된다.
   <br>
2. 변수의 우선순위를 시각화 한다.[^3]

|       | 선택지1 | 선택지2 | ... |
| :---: | :-----: | :-----: | :-: |
| 변수1 |         |         |     |
| 변수2 |         |         |     |
|  ...  |         |         |     |

|             | 자전거1 | 자전거2 | 자전거3 |
| :---------: | :-----: | :-----: | :-----: |
| 허리 편안함 |   네    | 아니오  | 아니오  |
|    가격     |   450   |   350   |   700   |
|   디자인    | 아니오  |   네    |  최상!  |

<br>

3. 장기적으로 자신에게 이로운가?
   <br>
4. 결정을 미루거나, 후회하지 않고 표를 사용한 결정을 존중하여 우유부단함을 줄여야 한다.

[^1]: 선택지들이 고만고만해서 고르기 어렵다. 뇌가 미세한 차이점을 찾으려다가 결정의 시기를 놓치기도 한다.

[^2]: 이것이 회피임을 받아들여야 한다.

[^3]: 그러지 않으면 방향을 잃고 뇌에 휩쓸리게 된다.
